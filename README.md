📊 Sales Overview Dashboard — Power BI

A regional sales performance dashboard built in Power BI, comparing Current Year (CY) vs Previous Year (PY) performance across four U.S. regions with drill-down detail by state.

📊 Overview

A single-page, highly interactive dashboard covering four regions — Central, East, South, and West — with side-by-side performance comparisons and a geographic breakdown.

🔑 Key Metrics Tracked (per region)


Total Sales (dynamic KPI title)
Prior-Year KPI Sales
Monthly sales trend


📈 Visuals & Analysis


Regional KPI cards — dynamic titles and current vs. prior-year sales for each region
Monthly trend charts — clustered column charts showing sales trend by month per region
Geographic map & bar chart — sales performance broken down by state/province and region
CY vs PY comparison table — pivot table comparing Sales, Profit, and Quantity across Current Year and Previous Year, with Year-over-Year (YoY) % change for each metric
Interactive slicers — filter by metric and by year


💡 Key Insights


Enables side-by-side comparison of regional performance at a glance
Highlights YoY growth/decline in Sales, Profit, and Quantity by region
Surfaces top and bottom performing states via the geographic map


🛠️ Tools Used


Power BI Desktop — data modeling, DAX measures (including dynamic titles and YoY calculations), report design
SQL — source data extraction and transformation


📁 Files


Sales_Overview.pbix — full Power BI report file
screenshots/ — exported image of the dashboard


📝 Note

Year-over-Year totals are averaged across all regions. For years without prior-year data (e.g., 2021 in this dataset), PY figures display as "No Data" by design.

🚀 How to Explore

Download Sales_Overview.pbix and open it in Power BI Desktop (free) to interact with the slicers and explore regional detail.
